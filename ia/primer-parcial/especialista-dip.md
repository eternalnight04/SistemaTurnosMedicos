## Reflexión personal

El apoyo de Copilot Agent fue útil para agilizar la elaboración de ejemplos y la redacción estructurada del análisis. Aun así, resultó clave revisar y ajustar cada propuesta para que esté alineada con los requisitos y el dominio real del sistema de turnos médicos. Este proceso ayudó a profundizar no solo en el significado del DIP, sino en cómo implementarlo efectivamente para mejorar la escalabilidad y mantenibilidad del software. Considero valioso este enfoque para futuros proyectos.
