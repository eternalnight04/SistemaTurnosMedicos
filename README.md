## Introducción

### Breve descripción del proyecto:

Este proyecto tiene como objetivo desarrollar el diseño orientado a objetos de un sistema para la gestión de turnos de un consultorio médico.

El trabajo parte del analisis de información deliberadamente dispersa (mails, audios, notas y transcripciones), a partir de la cual se identifican requisitos funcionales y no funcionales, actores, casos de uso y un boceto inicial de clases, aplicando los fundamentos de la Programación Orientada a Objetos.
